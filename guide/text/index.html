Defaults

You don't need to use the specific classes above all the time if you just want basic styles. Add the class dash-text to any element and basic styles will be applied to its children. 

    <div class="dash-text">
      <h4>A Heading</h4>
      <p>Some body text. And <a>a link</a>.</p>
    </div>
