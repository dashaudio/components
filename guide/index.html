Typography & Colours

General

The Dash corporate font is Proxima Nova, mostly used in weights 100, 400, 800 (Thin, Regular and Extrabold).

Headers

Headers are set in extrabold, generally going from 44px to 32px, 22px and 18px and are always black or white.

Different sizes may be used for one-off occasions, like the larger-than-life type above. In that case make them sufficiently different to the standard sizes.

Thin

You may also use Proxima Nova Thin if the overal page starts to look too heavy, but only in sizes 32 and above, like in the header to this block.

All Caps

Make sure type set in all caps also has a letter spacing of 0.1em.

Paragraph

General paragraph copy is set in Proxima Nova Regular, 16px and in medium grey—#626262

Links within a text block are black and blue—#5284FF—on hover.

Make sure to use glyphs appropriately. Use em-dashes, apostrophes, semi-colons etc.

Make sure to use glyphs and ligatures (em-dashes, apostrophes, semi-colons etc.) appropriately. More on this here. Btw, footnotes are set in 12px Proxima Nova Regular

They can also be a lighter grey
